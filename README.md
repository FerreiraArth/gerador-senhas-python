# 🔒 Gerador de Senhas Python

Projeto gerador de senhas em python para criar senhas mais seguras

---

## 🚀 Funcionalidades

* Criar senhas fortes e aleatória 
* Implementar símbolos
* Fazer suas senhas mais seguras

## 🛠️ Tecnologias

* Python
* random / secrets
* strings
